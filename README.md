# **Kahoot**
## **Descriere**
Proiectul reprezintă o platformă interactivă de tip quiz destinată testării cunoștințelor într-un mod dinamic și atractiv.
Aplicația permite crearea, administrarea și participarea la sesiuni de întrebări cu răspunsuri multiple, desfășurate în timp real.

Utilizatorii pot genera seturi de întrebări personalizate, pot gestiona participanți și pot vizualiza scorurile obținute în urma fiecărei sesiuni.
Interfața este intuitivă, asigurând o experiență ușor de utilizat atât pentru organizatori, cât și pentru jucători.
